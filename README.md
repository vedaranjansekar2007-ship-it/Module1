EX NO 05(e)

## Aim:
Write a Python program to check whether the entered year is 20th century is valid or not using if..else statement

## Algorithm:
Step 1: Get the input as a year
Step 2: Check the given year is divided by 4.
Step 3: If divided then print Year is 20th Century.
Step 4: If not divided then print Year is NOT 20th Century.

## Program:
```
year = int(input())
if year >= 1900 and year <= 2000:
    print("Year is 20th Century.")
else:
    print("Year is NOT 20th Century.")
```
